****************************In the Name of God****************************

The Toolbox is tested with MATLAB R2015a and Octave-4.0.0 in Windows and Linux.

	To install the Toolbox simply unpack the archive.

	Adjust your MATLABPATH to include DeeBNet.

	Download tested datasets (or use prepareMNIST_Small function in MNIST demos).

		MNIST

		ISOLET

		MATLAB version of the 20 Newsgroups data set

	Run the demo scripts to see what it can do (change dataset path in scripts). See the documentations for more details.

		test_classificationMNIST.m
		test_classificationISOLET.m
		test_classification20Newsgroups.m
		test_generateDataMNIST.m
		test_getFeatureMNIST.m
		test_plotDataMNIST.m
		test_reconstructDataMNIST.m
		test_sparsityAndBasesFunctionMNIST.m
		…
